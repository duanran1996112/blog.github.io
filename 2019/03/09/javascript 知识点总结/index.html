js 中知识点总结

原型链链的指向

解析：a 上没有 a、b 方法，通过 a.proto（A.prototype）上他的原型链上查找（无），通过 a.proto.proto（Object.prototype）上查找发现 b 方法，所以调用 a 方法报错，b 方法输出 2。A 上没有 a、b 方法，通过 A.proto（Function.prototype）上他的原型链上查找发现 a 方法，输出 1，通过 A.proto.proto（Object.prototype）上查找发现 b 方法，,输出 2。

函数作用域

解析：函数会产生作用域，如果当前作用域没有某一个变量，会去他的上级作用域查找。函数的上级作用域为他声明时的上级，并非调用时的上级。

nodejs 事件队列

解析：nodejs 中事件队列中 setTimeout 属于 timer 队列，setImmediate 属于 check 队列，timer 队列执行顺序优于 check 队列。
