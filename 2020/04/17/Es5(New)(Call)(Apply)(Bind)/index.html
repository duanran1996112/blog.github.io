Es5 New Call Apply Bind

New

New 做了什么：

  1. 创建一个新的对象
  2. 这个对象会的 /_/proto//_ 属性会与这个构造函数的 prototype 进行链接
  3. 这个对象会被绑定到调用函数的 this
  4. 如果这个构造函数没有返回 object 类型，那么会返回这个新对象，否则返回 return 的内容

Bind

说明：可使用 call、apply 进行封装
