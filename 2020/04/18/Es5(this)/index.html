Es5(this)

This 指向

This 的指向与级别优化，级别高的覆盖级别低的

一级

函数中 this 指向 window，使用 ‘use strict’ 时 this 指向 undefined

二级

谁调用函数函数的 this 就指向谁

三级

call 与 apply 可以改变函数中 this 的指向

四级

bind 可以改变函数中 this 的指向（但不会立即执行）

五级

new 可以把 this 绑定在一个新对象上

六级

箭头函数中没有自己的 this（无法当做构造函数），this 为（父级或祖先）最靠近箭头函数的普通函数作用域中的 this
