Es6新增数据结构

前述

所谓数据结构，就是存储数据的一种方式

Es6新增加了 set 与 map 这两种数据结构

Set

Set 数据结构的特点

特点：set 数据结构没有重复

Set 数据结构的声明

传递一个数组

结合 set 数据结构的特点与声明，我们来看一下下面这些例子：

Set 数据结构上的方法

set.add()

为 Set 数据结构添加某个值,返回 Set 结构本身

set.delete()

为 Set 数据结构删除某个值,返回一个布尔值，表示删除是否成功

set.has()

表示 Set 数据结构是否含有某个值,返回一个布尔值，表示该值是否为 Set 的成员

set.clear()

为 Set 数据结构清除所有值,没有返回值

Map

Map 数据结构的特点

特点：map 数据结构，可以接受任何类型的值，作为键值

它类似于对象，也是键值对的集合，但是“键”的范围不限于字符串，各种类型的值（包括对象）都可以当作键。

Map 数据结构的声明

传递一个二维数组

Map 数据结构上的方法

map.size

表示 Map 数据结构的长度

map.set()

为 Map 数据结构添加属性

map.get()

获取 Map 数据结构某个属性的属性值，返回值是这个属性的属性值

map.has()

判断 Map 数据结构是否有该属性，返回值是布尔值

map.delete()

删除 Map 数据结构的某个属性，返回值是布尔值

map.clear()

删除 Map 数据结构的所有值
